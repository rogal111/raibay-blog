Python dogania Javę w najnowszych rankingach języków programowania

Po 20 latach na szczycie list najpopularniejszych języków programowania, Java może zostać zdetronizowana przez Pythona.

Jak wynika z październikowego rankingu TIOBE, popularność Javy w ostatnim miesiącu stale malała, co daje Pythonowi szanse na zajęcie miejsca Javy, jako drugiego najbardziej popularnego języka programowania na świecie. Jeżeli Python wyprzedzi Javę, będzie to pierwszy raz od powstania rankingu TIOBE w 2001 roku, kiedy to Java znalazłaby się poza dwoma czołowymi miejscami.

Jak zauważa prezes TIOBE, Paul Jansen, C i Java od dwóch dekad konsekwentnie zajmują dwa pierwsze miejsca rankingu. Jednak obecnie 25-letnia Java bardzo traci na popularności, czego dowodem jest spadek o 4,32 procent w porównaniu z wynikami z października 2019.  

Jansen zwrócił uwagę na pogarszającą się sytuację Javy już we wrześniu, wskazując na spadek o 3,81 punktów procentowych w porównaniu do roku poprzedniego. Natomiast popularność Pythona stale rośnie, ze względu na wykorzystanie tego języka przez programistów z dziedziny data science, a także jego zastosowanie w uczeniu maszynowym.

Indeks TIOBE mierzy popularność języków programowania na podstawie wyszukiwań danego języka w 25 wyszukiwarkach. Oprócz TIOBE dostępne są również rankingi tworzone m.in., przez IEEE Spectrum, RedMonk, GitHub, Stack Overflow, a każdy z indeksów wykorzystuje inną metodologię, przez co rankingi nie zawsze się pokrywają. Wygląda jednak na to, iż październikowy indeks TIOBE potwierdza to, co RedMonk zaobserwował w swoich rankingach z lipca 2020 roku. Rankingi RedMonk opierają się na danych z GitHub i Stack Overflow.

RedMonk konsekwentnie umieszcza JavaScript na pierwszym miejscu, a do niedawna drugie miejsce należało do Javy. Wszystko zmieniło się w lipcu tego roku, kiedy to Python zajął jej miejsce w rankingu. Dane udostępnione przez TIOBE pokazują, że popularność Javy wynosi obecnie 12,56%, natomiast Pythona 11,28, co daje różnicę zaledwie 1,3 punkta procentowego między tymi językami.

Stephen O'Grady z RedMonk uważa, iż Java nadal pozostanie ważnym językiem, jednak jej popularność jako “języka pierwszej potrzeby” jest zagrożona, ze względu na rosnące zainteresowanie programistów innymi językami.

W raporcie zleconym przez Oracle zauważono, że konieczne jest wprowadzenie innowacji w Javie, aby odciągnąć potencjalnych programistów od nowszych języków, takich jak Rust i Kotlin. Niemniej jednak, Oracle zwraca uwagę, że język Java jest używany przez 69% pełnoetatowych programistów na całym świecie.

Java została stworzona w 1995 roku przez Jamesa Goslinga w Sun Microsystems. Gosling i inni inżynierowie Sun zdecydowali się stworzyć Javę, ze względu na zwiększone zastosowanie procesorów w telefonach komórkowych, windach, lokomotywach, systemach kontroli procesów i sprzęcie audiowizualnym. W latach 1990-1991, po wycieczkach po Japonii i Europie w celu odwiedzenia znanych marek elektroniki użytkowej, Gosling zauważył, że wiele firm popełnia te same błędy, które przemysł komputerowy popełniał 20 lat wcześniej i postanowił stworzyć język, który teraz jest tak popularny na całym świecie.

10 najpopularniejszych języków w październikowym indeksie TIOBE to C, Java, Python, C++, C#, Visual Basic, JavaScript, PHP, R i SQL.

Co myślicie o wynikach październikowego rankingu? Jesteście #teamjava czy #teampython?
